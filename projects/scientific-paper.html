Changes in Fine Particulate Matter (PM2.5) Levels in the United States

Project Overview

This project represents my first attempt at drafting a scientific paper using LaTeX. The main objective was not to produce a fully rigorous research article, but to learn and practice the process of scientific writing, data analysis, and formatting according to journal standards.

Key activities and skills developed:

  • LaTeX & Overleaf: Learned to structure a scientific paper using the Elsevier elsarticle template.
  • Bibliography Management: Exported references from Zotero via the BibTeX plugin and integrated them into .bib files in Overleaf.
  • Data Analysis in R:
    • Applied bootstrap methods to create confidence intervals when standard assumptions were not met.
    • Calculated confidence intervals for median comparisons and differences instead of relying solely on traditional statistical tests.
    • Used both the replicate & sample functions in base R and the bootstrap package.
  • Figures & Tables: Inserted analysis results, tables, and figures into the LaTeX document.
  • Journal Formatting: Practiced adapting content to specific journal requirements using pre‑existing data and results.

While the literature review and results interpretation may not follow strict scientific rigor, this project served as a valuable opportunity to explore the full workflow from analysis to formatted manuscript.

Infant and Juvenile Mortality Analysis

Context

This project provides a comprehensive survival analysis of infant and juvenile mortality using a fertility survey dataset. The primary goal is to identify key demographic and socio-economic factors influencing child survival, employing Kaplan-Meier curves for visualization and Cox Proportional Hazards models for multivariate analysis.
